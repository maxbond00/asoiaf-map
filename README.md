# ASOIAF Interactive Map

An interactive map, character relationship graph, historical timeline, and fan theory tribunal for *A Song of Ice and Fire* by George R.R. Martin.

**Live site:** https://maxbond00.github.io/asoiaf-map/

---

## Features

- **Map** — Track character locations across all five books with book-by-book filters
- **Relations** — Interactive force-directed character relationship graph with house filters
- **History** — Illustrated timeline of major ASOIAF eras from the Dawn Age to the War of the Five Kings
- **Theories** — 55 popular fan theories with write-ups; vote Canon / Likely / Unsure / Unlikely / No Way and see results sorted by fan consensus

---

```bash
npx serve .

```
